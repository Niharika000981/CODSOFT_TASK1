# 📝 Python To-Do List Application

A simple and user-friendly **To-Do List desktop application built with Python and Tkinter**.  
This project helps users manage daily tasks by allowing them to add, update, complete, and delete tasks through a graphical interface.

It is a beginner-friendly project that demonstrates **Python GUI development**, event handling, and task management logic.

## 🚀 Features

- ➕ Add new tasks
- ✏️ Update existing tasks
- ✔ Mark tasks as completed
- ❌ Delete tasks
- 📋 Task list displayed in a table format
- 🖥 Simple and clean GUI interface

## 🛠 Technologies Used

- Python
- Tkinter (Python GUI Library)
